# projeto-integrador-extensao-saiba-mais-servico-publico
🏛️ Saiba Mais Serviço Público: Projeto de extensão que utiliza IA para simplificar o acesso a serviços públicos. Busque um serviço e receba o passo a passo completo, requisitos e documentos necessários para garantir seu acesso. Tecnologia a serviço da cidadania para combater a burocracia.
