# Saiba Mais Serviço Público 🏛️🤖
## Facilitando o acesso à cidadania através da Inteligência Artificial.

O **Saiba Mais Serviço Público** é um projeto integrador de extensão que busca reduzir a barreira de informação entre o cidadão e seus direitos. A aplicação utiliza IA para explicar de forma simples e direta como funcionam diversos serviços públicos, informando os requisitos, documentos e o passo a passo necessário para acessá-los.

# 🚀 O Problema
Muitas vezes, o cidadão sabe que tem um direito ou precisa de um serviço, mas se perde na burocracia, termos técnicos ou sites governamentais complexos. A falta de informação clara é um dos maiores obstáculos para a justiça social e o exercício da cidadania.

# ✨ Funcionalidades
* **Busca Inteligente:** O usuário pode pesquisar por serviços em linguagem natural (ex: "Como faço para tirar minha primeira habilitação?").

* **Guia de Acesso:** Explicação detalhada do que é necessário para solicitar o serviço.

* **Checklist de Documentos:** Lista clara dos documentos exigidos para evitar idas e vindas desnecessárias.

* **Simplificação de Termos:** Tradução de termos jurídicos ou burocráticos para uma linguagem acessível.
